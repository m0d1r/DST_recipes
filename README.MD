# DST Блюда — веб-справочник для Don’t Starve Together

Семестровый проект. Приложение на чистом JavaScript получает данные из публичного HTTP-API вики Don’t Starve, отображает список блюд с характеристиками (здоровье, голод, рассудок), позволяет сортировать, фильтровать по категориям, искать по названию и смотреть статистику.

## Выполненные требования

1. Исходный код в Git-репозитории: репозиторий публичный, коммиты сделаны.
2. Источник данных – публичное HTTP-API: MediaWiki API вики Don’t Starve (https://dontstarve.fandom.com/api.php), запросы через fetch.
3. Структура файлов: index.html, styles.css, js/ с подпапками data и ui.
4. Модульность кода: разделение на слой данных (recipesApi.js) и слой отображения (uiRenderer.js), связка через main.js.
5. Экран приветствия: форма с полем «Имя» и кнопкой «Далее».
6. Приветствие в шапке: после входа выводится «Привет, имя!».
7. Отображение данных: карточки блюд с названием, здоровьем, голодом, рассудком.
8. Элементы управления: поле поиска, select для сортировки (поле + порядок), вкладки для фильтрации.
9. Обработчики событий: click, change, input.
10. Сортировка: по названию, здоровью, голоду, рассудку (возрастание/убывание).
11. Фильтрация: по категориям (все блюда / здоровье / голод / рассудок).
12. Статистика: общее количество блюд, средние значения здоровья/голода/рассудка, количество по категориям.

## Архитектура

Модель-Представление-Контроллер на ES6 модулях:
- data/recipesApi.js – модель (fetch, парсинг, сортировка, фильтр, статистика)
- ui/uiRenderer.js – представление (экран приветствия, карточки, события)
- main.js – контроллер (импорт модулей, передача колбэков)

## Установка и запуск

1. Клонировать репозиторий:
   git clone https://github.com/m0d1r/DST_recipes.git
   cd DST_recipes
2. Запустить локальный веб-сервер (Live Server в VSCode).
3. Открыть приложение в браузере.

## Использование

- На экране приветствия ввести любое имя и нажать «Исследовать».
- В шапке появится приветствие.
- Загрузятся карточки блюд из вики Don’t Starve.
- Выбрать сортировку (поле и порядок).
- Нажать вкладку для фильтрации по категории.
- Ввести текст в поле поиска для фильтрации по названию.
- Статистика обновляется автоматически.
- Кнопка «Сменить имя» возвращает к экрану приветствия (имя не сохраняется).
